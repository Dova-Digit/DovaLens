# DovaLens – Automated Data Profiling & Drift Detection

DovaLens è un motore automatico di data analysis progettato per:

- Profilazione dataset  
- Analisi statistiche avanzate  
- Drift detection (KS-test)  
- Clustering automatico  
- Rilevamento anomalie  
- Generazione report HTML  

## Installazione

```bash
pip install .
```

## Utilizzo

```bash
dovalens path_al_tuo_dataset.csv
```

## Output

```
report.html → report completo con analisi, statistiche e anomalie.
```
